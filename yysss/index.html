Note 1: 同时在结界中养育多个相同式神可同时获得传记中的经验增加
例如，在结界中同时养育5个萤草，每个萤草获得了10000点经验， 那么传记中将显示“通过结界养育获得50000点经验”。

Note 2: 参与阴阳寮的结界突破也会计算在“结界突破进攻次数”中

Note 3: 技能升级次数可以通过反复喂养来更快的增加
例如，有三个萤草A、B、C，将萤草B和萤草C喂给萤草A，传记中技能升级次数是2； 将萤草C喂给萤草B，再将萤草B喂给萤草A，传记中技能升级次数是3。

Note 4: 斗技中式神对抗取胜数可以用好友切磋代替
像“犬神在斗技中取得5次对抗九命猫的胜利”这样的，可以让好友带上九命猫进行切磋，然后就能轻易刷够5次了。
但是，像“参与30场斗技并取胜”这样的就必须去斗技中打了。
